Nödutgångar


- Att veta vid nödsituationer
- Placeringar dörrar
- Placering brandsläckare
- Placering gas

Belysning


- Viktiga att tända/släcka vid öppning/stängning, placering med bilder?
- Belysning dag/kväll (ljuslyktor bord, biljardbord vip, servering)

Dagskassa


- Skilja på kassa bar dag/bar 1/plånbok
- Guide med bilder till rätt räknesätt för både kontrollräkning och efter stängning

Trivec


- Sidor att arbeta utifrån arbetssektioner (bar, biljardkassa, servis, garderob)
- Vad man gör vid presentkortsbetalning, felslag, retur till gäst, dela upp nota, nekat kort, springnota osv
- Tekniska problem vs lösningar. Starta om kassaskärm, starta om dino, osv

Dagliga rutiner


- Veckodagsbaserat utifrån arbetsområde, krav på att det ska göras, arbetssätt från start till slutet av pass, vad man prioriterar om tiden inte räcker till
- Rutiner som görs månadsvis/årsvis
- Vem gör vad? Tydliga ansvarsområden men fortsatt samarbete mellan personal vid behov

Övergripande rutiner


- Öppning veckodag/helgdag
- Stängning veckodag/helgdag
- Preppa för kväll/öppning dagen efter

Gästens upplevelse


- Bemötande vid dörr: välkomna, placera, servera
- Uppvaktande: uppmärksamhet gentemot gäst, hålla övergripande kontakt med hela serveringen
- Avsluta middagen: skicka vidare till aktivitet? förklara processen från betalning till uthyrning av biljardbollar/shuffle etc.

Rekommendationer/försäljning


- Att kunna menyn/veta vad som smakar vad och därmed föreslå för gästen efter dens tycke
- Merförsälja; tex dipp vid burgare, fler smårätter, trycka på god efterrätt om gästen tex börjar med huvudrätt

Missnöjd gäst


- Vad gör man om gäst är missnöjd? Delvis frihet under ansvar men de flesta fall se över situationen med ansvarig på plats - ge exempel och underlag
- Kompensation - att hålla koll på alkohollagen, lämplig procentsats rabatt efter bekräftelse av ansvarig, beteende & jargong från personal


Berusad gäst


- Hur avgör man att gäst druckit för mycket? Hänvisa till Houmans utbildning/sammanfatta utbildningsdagarna
- Att neka gäst, att inte överservera osv

- Hur startar man en match? Snabb guide steg för steg, byta kanal på box -> välja rätt box på rätt projektor
- Vilken match ska visas var? Att veta vilken som har mest publik. börja med veckoscheman för vart matcherna visas?
- Tekniska svårigheter, att kunna lösa innan man kontaktar ansvariga för hjälp. Vikten av att vara ute i god tid och preppa för stormatcher!
- Lathund över kanalerna, siffror vs kanaler

Meny- och dryckesguide


- Krav på att personal ska kunna menyn (främst mat), vad rätterna innehåller, allergener & ingredienser
- Att ta hänsyn och fråga gäster om allergier eller andra dietära situationer

Prislista aktiviteter


- Priset per aktivitet
- Att hålla koll på olika priser utifrån tider vs. medlem/icke-medlem

Bokningar


- Hur man för in bokning i kalendern/häftet: krav på bokningens namn, antal pers, tid, vad sällskapets huvudaktivitet är, telefonnummer samt slutligen signering av personal som tagit emot
- Veta hur man styr sällskap i rätt riktning, tex mycket vid viss tid, rikta om gästen att boka vid senare tid?
- Veta vad man får/inte får ta bokning på

Personalkollen


- Snabb guide, schemat röd/grönmarkering

Telefonlista


- Viktiga nummer både personal/leverantörer i pärm
- Sekretess, inte ge ut nummer osv.
